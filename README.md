# Student Performance Analytics Dashboard

## Project Overview

The Student Performance Analytics Dashboard is an interactive web application developed using Python, Dash, Plotly, and Pandas. The dashboard combines multiple student datasets and provides visual insights into academic performance, attendance, study habits, and departmental statistics.

## Features

* Interactive Dashboard
* Department Filter
* Gender Filter
* KPI Cards
* Department Performance Analysis
* Attendance vs Marks Analysis
* Study Hours vs Marks Analysis
* Department Distribution Visualization

## Technologies Used

* Python
* Pandas
* Plotly
* Dash

## Dataset Information

### Dataset 1

Student Academic Performance Dataset

Fields:

* Student ID
* Gender
* Department
* Attendance
* Internal Marks
* Final Marks

### Dataset 2

Student Activity Dataset

Fields:

* Student ID
* Sports Participation
* Club Participation
* Study Hours
* Assignment Completion

## Dashboard Components

### KPI Metrics

* Total Students
* Average Marks
* Average Attendance
* Assignment Completion
* Top Performing Department
* Best Attendance Department

### Visualizations

* Department-wise Marks Analysis
* Department Distribution Pie Chart
* Attendance vs Marks Scatter Plot
* Study Hours vs Marks Scatter Plot
